# Fayvrs Ad Mockup Export Guide

## Quick Screenshot Instructions

### For High-Quality Static Images

1. **Open in Browser**
   - Navigate to `/mockups/ads/` to see all screens
   - Click individual screen to view fullscreen

2. **Set Mobile Dimensions**
   - Open Developer Tools (F12)
   - Toggle device toolbar (Ctrl+Shift+M)
   - Set dimensions: **390 x 844** (iPhone 14) or **393 x 852** (Pixel 7)
   - Or custom: **1080 x 1920** for highest quality

3. **Capture Screenshots**
   - Use browser screenshot tool (F12 > ... > Capture screenshot)
   - Or use Snipping Tool / macOS Screenshot (Cmd+Shift+4)
   - Save as PNG for best quality

### For Screen Recording (Video Ads)

**Recommended Tools:**
- **Windows**: OBS Studio (free), Camtasia
- **macOS**: QuickTime Screen Recording, ScreenFlow
- **Online**: Loom, CloudApp

**Recording Settings:**
- Resolution: 1080 x 1920 (9:16 vertical)
- Frame rate: 30 fps minimum (60 fps ideal)
- Format: MP4 (H.264 codec)

### Video Editing Workflow

1. **Import Screenshots** into video editor
2. **Duration**: 2-3 seconds per screen
3. **Transitions**: Quick fade (0.3s) or swipe
4. **Add voiceover** (see AD_STORYBOARD.md for scripts)
5. **Background music**: Upbeat, royalty-free tracks
6. **Export**: 1080x1920, MP4, 30fps

### Recommended Screen Sequence

**15-Second Ad:**
1. Hero Splash (2s)
2. Problem Screen (3s)
3. Live Matching (3s)
4. Success Screen (3s)
5. Download CTA (4s)

**30-Second Ad:**
Use screens 1-7 (Hero → Video Call)

**60-Second Ad:**
Use all 14 screens in order

## Export Checklist

- [ ] All screens captured at correct dimensions
- [ ] Images saved in high resolution (PNG format)
- [ ] Screens organized in narrative order
- [ ] Ready for video editor import
- [ ] Voiceover script prepared
- [ ] Music track selected

## Tips

✓ Use consistent timing for professional look
✓ Add subtle zoom animations in editor
✓ Keep text on screen long enough to read
✓ Test on actual mobile device before publishing
✓ A/B test different sequences for best performance
